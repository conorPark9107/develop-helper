정확한 시장 가격을 얻는데 기여하는 방법.

먼저 기여하는데 관심 가져주셔서 감사합니다.🙏

먼저 정확한 시장 가격을 위해 AODP(Albion Data Project)프로그램이 필요합니다!

- 알비온 온라인에서는 유저들에게 API를 제공하지 않지만, 네트워크 패킷을 보고 분석해서 저장하는 프로그램이 존재합니다. 이 프로그램이 AODP(Albion online data project)입니다.

- 이 홈페이지 개발자도 현재 이 프로그램을 사용하고 있는데 이 프로그램은 큰 리소스를 잡아먹지 않는것 같습니다.
(이 프로그램은 백그라운드에서 실행되고 저는 실행중인 상태로 zvz도 합니다.)

이 프로그램 사용하면 정지당하는거 아닌가요?

- 아니요. Sandbox Interactive 기술 책임자로부터 네트워크 패킷을 서비스를 방해하지 않는 선에서 보고 분석하는 것이라면 허용한다는 답변을 받은 기록이 있습니다.
(답변 링크)

- 또한 제가 개발한 것이 아니며 오픈 소스입니다. 이 프로그램이 안전한지에 대해서 직접 확인 해보고 싶다면 (링크) 를 클릭하여 소스 코드가 안전한지 직접 확인해보실수 있습니다.

다운로드 방법.

- 정확한 시장 가격을 얻기 위해서는 먼저 AODP github Releases에서 프로그램을 다운로드 해야합니다.

- 아래 이미지에서 검정 박스 부분을 보시고 자신의 컴퓨터의 운영체제에 맞게 다운로드 합니다.
(예를들면 윈도우를 사용한다면 albiondata-client-amd64-installer.exe 파일을 다운로드)

- 다운로드한 파일을 설치하기 위해 실행하는데 다음과 같은 메세지가 발생합니다.

- 아래 이미지 검정 박스 부분을 클릭하면 실행 버튼이 나오는데 클릭하여 실행합니다.

- 설치중간에 WinPcap 인스톨러가 등장하는데, 전부 기본값으로 넘기며 설치합니다.


- 더이상 설치 메세지가 나오지 않는다면 설치가 완료된 것입니다. 다음 재부팅을 하시면 백그라운드에서 프로그램이 구동되는것을 확인하실 수 있습니다.
- 종료는 마우스 우클릭하여 종료 시킬 수 있습니다.
(아래 이미지 참고)

- 아래와 같이 백그라운드로 구동되는것을 확인 하였다면 설치가 완료된 것입니다. - 설치가 완료된 후 부터 알비온 시장에서 아이템을 검색하여 가격을 조회하거나 할때마다 이 홈페이지에서 시장가격이 실시간으로 최신화됩니다.

제거하는 방법.

- 제어판에 들어가서 Albion Data Client를 검색한 후 제거하시면 됩니다.
